# 로또
## 1️⃣ 진행순서

- [ ] 로또 구입 금액을 입력받는다.
- [ ] 발행 가능한 로또 수량을 출력한다.
- [ ] 입력된 금액만큼 로또를 발행한다.
- [ ] 발행한 로또 번호를 출력한다.
- [ ] 당첨 번호와 보너스 번호를 입력 받는다. 
- [ ] 발행한 로또와 당첨 번호 및 보너스 번호를 비교하여 각 등수마다 당첨된 개수를 계산한다.
- [ ] 당첨 내역을 출력한다.
- [ ] 수익률을 계산한다.
- [ ] 수익률을 출력한다.

## 2️⃣ 기능목록
- [ ] 입력받기
  - [ ] 로또 구입 금액
  - [ ] 당첨 번호
  - [ ] 보너스 번호
- [ ] 결과출력
  - [ ] 발행 가능한 로또 수량
  - [ ] 당첨 내역
  - [ ] 수익률
- [ ] 계산하기
  - [ ] 당첨 번호와 보너스 번호 비교
  - [ ] 수익률
